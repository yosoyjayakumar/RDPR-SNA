# RDPR-SNA

A web-based portal providing quick access to various websites and services related to the Rural Development and Panchayat Raj (RDPR) department and other relevant entities. This project is designed to streamline access to frequently used online resources for government officials and staff.

## Features

*   **Centralized Dashboard:** A single page with a collection of links to important websites.
*   **Categorized Links:** Websites are grouped into logical sections for easy navigation (e.g., Featured, Scheme Side, etc.).
*   **Quick Search:** A search bar to filter and find specific websites quickly.
*   **Responsive Design:** The layout adapts to different screen sizes, making it accessible on desktops, tablets, and mobile devices.
*   **Visit Counter:** Tracks the number of times the page has been visited.
*   **Animated Background:** A visually appealing animated background using HTML5 Canvas.

## Technology Stack

*   **HTML5**
*   **CSS3**
*   **Vanilla JavaScript**
*   **CounterAPI.dev:** For tracking page visits.

## How to Use

1.  Clone the repository:
    ```bash
    git clone https://github.com/yosoyjayakumar/RDPR-SNA.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd RDPR-SNA
    ```
3.  Open the `index.html` file in your preferred web browser.
